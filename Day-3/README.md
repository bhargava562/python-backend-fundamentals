# 🛒 E-commerce SQL Project

## Overview
This project demonstrates SQL concepts including DDL, DML, Joins, Aggregations, Transactions, and Database Design.

## Database Schema
Tables:
- users
- products
- orders
- order_items

## Relationships
- One User → Many Orders
- One Order → Many Order Items
- Many Products ↔ Many Orders

## Features Covered
✔ DDL & Constraints  
✔ DML operations  
✔ Complex joins  
✔ Aggregations  
✔ Subqueries & CTEs  
✔ Transactions  
✔ Normalization  

## How to Run
1. Run `schema.sql`
2. Run `seed_data.sql`
3. Run remaining practice files
